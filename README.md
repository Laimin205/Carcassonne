Проект по созданию цифровой версии классической настольной игры [Каркассон](https://clck.ru/3T3h9d) ([правила](https://clck.ru/3T3hAN)) для 2-5 игроков с поддержкой базовых механик и отдельных дополнений.

---
## Распределение фичей по итерациям
### Итерация 1
- [ ]  Отображение и движение игрового поля
- [ ]  Логика размещения тайлов
- [ ]  Базовая обработка созданных во время игры объектов - города, дороги, монастыри
- [ ]  Своевременный подсчёт очков

**Результат итерации:** базовая играбельная версия для одного игрока без механики крестьян

### Итерация 2
- [ ]  Крестьяне и поля
- [ ]  Мультиплеер и игровая статистика
- [ ]  Логика конкуренции за объекты и их объединения
